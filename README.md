# Classy Containers

Premium residential dumpster / hook lift container rental serving Park City and the Wasatch Back, Utah.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- CSS-in-JS (styles embedded in component)

## Service Area

Park City · Heber · Midway · Kamas · Jeremy Ranch · Deer Valley · Snyderville · Wasatch County
